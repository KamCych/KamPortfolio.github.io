    • Space Defense


    Space Defense is made with the C++ SDL2 library, inspired by Metroid. Using pathfinding, enemies slowly float towards the power supply.






    Development

    This project was my first real experience with C++, it taught me all the key features of the language, and an even deeper understanding of how games are made.

    Features:

  • AI pathfinding
  • Tilemaps
  • Tilesheet animation
  • Collision
  • Game states - menu, game
  • Object Oriented Programming
