    • Space Defense


    Space Defense is a game made with the SDL2 C++ library, inspired by Metroid. Using pathfinding, enemies slowly float towards the power supply.






    Development

    Features:

  • AI pathfinding
  • Tilemaps
  • Tilesheet animation
  • Object Oriented Programming

    • This project was my first real experience with C++, it taught me all the key features of the language, and even an even deeper understanding of how games are made.
