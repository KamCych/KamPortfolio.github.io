    • Flappy Dragon


    An endless arcade game, including fireballs and destruction! Exciting visual and audio effects, with increasingly difficult, random obstacles.






    Development

    Made in Godot, this project tackles key parts of game development; UI, animation, save system, object oriented programming, and more. This project introduced me to all of the mentioned topics.

    Features:

  • Animation
  • Saving data
  • Particles
  • VFX - screen shake
  • Audio
  • Object Oriented Programming
  • UI
  • Parallaxing backgrounds
